# Function Substitution
Proof of concept mod for function substitution.
Function substitution/injection allows for optimization and modding capabilities while maintaining a vanilla environment.

# Benchmarks
For this benchmark, #bs.math:isqrt of the [bookshelf math library](https://smithed.net/packs/bookshelf-math) was called 6000x times by a vanilla function.
